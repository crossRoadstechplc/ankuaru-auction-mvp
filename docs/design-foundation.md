# Ankuaru Design Foundation

## Brand Direction
Ankuaru is a professional B2B marketplace for specialty coffee and spices, featuring live auctions and sourcing feeds. The brand signifies trust, agriculture, and high-quality commerce.

- **Vibe**: Professional, Clean, Warm, Trustworthy, Modern Dashboard.
- **Industry Icons**: Coffee, Earth, Harvest, Marketplace.

## 4. Layout Rules

### Container Max-Width
- **Standard**: `1280px` (`max-w-7xl`)

### Horizontal Padding
- **Mobile**: `16px` (`px-4`)
- **Desktop**: `24px` (`px-6`)

### Component Sizing
- **Default Height (Buttons/Inputs)**: `40px`
- **Standard Card Padding**: `24px` (`p-6`)

## 5. Visual Primitives

### Radius Scale
- `sm`: 4px
- `md`: 6px
- `lg`: 8px
- `xl`: 12px

### Shadow Scale
- `sm`: Subtle elevations
- `md`: Standard card shadows
- `lg`: Focus/Modal elevations
